# Dashboard Pemantauan Ketimpangan Pendidikan Indonesia 2025

## Deskripsi
Project ini mengimplementasikan Medallion Architecture (Bronze, Silver, Gold) menggunakan Apache Spark, Airflow, MinIO, PostgreSQL, dan Superset.

## Arsitektur Pipeline

Dataset XLSX
↓
Python ETL
↓
MinIO Bronze
↓
PySpark Silver
↓
PySpark Gold
↓
PostgreSQL
↓
Superset Dashboard

## Teknologi
- Apache Spark
- Apache Airflow
- MinIO
- PostgreSQL
- Apache Superset
- Docker
